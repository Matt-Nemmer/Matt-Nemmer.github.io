Project Biomapper

A Mobile Map-App

Biomapper is a mobile application that displays geospatial data about tropical forest on a map. It is intended to assist forest ecologists in Africa with their work.

See Biomapper on Google Play Store

Back-end Server

The information displayed on the maps is stored in images on a back-end server. This server is an AWS EC2 instance that fulfills requests made by the mobile app.

Online User Guide

To help users understand how to work the app, the Biomapper User Guide was created. It is a website that explains how to perform specific tasks within the mobile app.
